# 📱 วิธีทำ Spendly ให้เป็นแอปบนมือถือ (ฟรี)

Spendly ตอนนี้เป็น **PWA** สมบูรณ์แล้ว — ติดตั้งลงมือถือเป็นแอปได้ ใช้ออฟไลน์ได้ ข้อมูลเก็บในเครื่อง

## ไฟล์ทั้งหมดในโฟลเดอร์นี้

| ไฟล์ | หน้าที่ |
|---|---|
| `index.html` | ตัวแอปหลัก (HTML + CSS + JavaScript) |
| `manifest.json` | ข้อมูลแอป (ชื่อ ไอคอน สีธีม) |
| `service-worker.js` | ทำให้ใช้ออฟไลน์ได้ |
| `data.json` | ข้อมูลเริ่มต้น (โหลดอัตโนมัติครั้งแรก) |
| `icon.svg` | ไอคอนแอป (ไฟล์ SVG ไฟล์เดียว ไม่มี .png แล้ว) |

> **ทางลัด:** `Spendly.html` คือไฟล์เดียวจบ — รวมทุกอย่าง (ไอคอน SVG ฝังในตัว) ดับเบิลคลิกหรือลากไป deploy ไฟล์เดียวได้เลย ไม่ต้องมีไฟล์อื่น
>
> ถ้าใช้เวอร์ชันแยกไฟล์ (`index.html`) ต้องอัปโหลด **ทุกไฟล์ไว้โฟลเดอร์เดียวกัน**

---

## ข้อมูลเก็บยังไง

- **อัตโนมัติ** — ทุกครั้งที่เพิ่ม/ลบรายการ ข้อมูลถูกบันทึกในเครื่องทันที (ปิดแล้วเปิดใหม่ไม่หาย)
- **ปุ่ม "ดาวน์โหลดไฟล์ JSON"** (หน้าตั้งค่า) — สำรองข้อมูลเป็นไฟล์ `.json` จริง เก็บไว้ หรือย้ายไปเครื่องอื่น
- **ปุ่ม "นำเข้าไฟล์ JSON"** — โหลดไฟล์ที่สำรองไว้กลับเข้ามา
- `data.json` ในโฟลเดอร์ = ข้อมูลตั้งต้น เปิดแอปครั้งแรกบนเครื่องใหม่จะดึงมาให้ (แก้ไฟล์นี้เองได้ถ้าอยากตั้งค่าเริ่มต้น)

## วิธี deploy ขึ้นเว็บฟรี — เลือกอันใดอันหนึ่ง

### วิธีที่ 1: Netlify Drop (ง่ายสุด ไม่ต้องสมัครก็ได้)

1. เปิด https://app.netlify.com/drop
2. ลาก **ทั้งโฟลเดอร์** (ที่มี index.html, manifest.json ฯลฯ) ไปวางในหน้าเว็บ
3. รอสักครู่ → ได้ลิงก์เว็บทันที เช่น `https://ชื่อสุ่ม.netlify.app`
4. เปิดลิงก์นั้นในมือถือ → ติดตั้งตามขั้นตอนด้านล่าง

### วิธีที่ 2: GitHub Pages

1. สมัคร GitHub → สร้าง repository ใหม่
2. อัปโหลดไฟล์ทั้งหมดเข้าไป
3. Settings → Pages → เลือก branch `main` → Save
4. ได้ลิงก์ `https://ชื่อคุณ.github.io/ชื่อrepo/`

### วิธีที่ 3: Vercel

1. สมัคร https://vercel.com → New Project → อัปโหลดโฟลเดอร์
2. Deploy → ได้ลิงก์ `.vercel.app`

> ทั้งสามแบบ **ฟรี** และให้ HTTPS อัตโนมัติ (PWA ต้องใช้ HTTPS ถึงติดตั้งได้)

## ติดตั้งลงมือถือ

### iPhone (Safari)
1. เปิดลิงก์เว็บใน **Safari**
2. กดปุ่ม **แชร์** (ไอคอนสี่เหลี่ยมมีลูกศรขึ้น)
3. เลื่อนหา **"เพิ่มไปยังหน้าจอโฮม"** → เพิ่ม
4. ได้ไอคอน Spendly บนหน้าจอ เปิดแล้วเต็มจอเหมือนแอป

### Android (Chrome)
1. เปิดลิงก์ใน **Chrome**
2. กดเมนู **⋮** มุมขวาบน
3. เลือก **"ติดตั้งแอป"** หรือ **"เพิ่มไปยังหน้าจอหลัก"**
4. ได้ไอคอนแอปบนหน้าจอ

---

## ลองในคอมก่อน (ไม่ต้อง deploy)

แค่ดับเบิลคลิก `index.html` ก็ใช้ได้เลย — แต่ฟีเจอร์ออฟไลน์ (service worker) กับการโหลด `data.json` อัตโนมัติ จะทำงานเฉพาะตอนเปิดผ่านเว็บ (http/https) ส่วนการเพิ่ม/ลบ/export/import ใช้ได้ปกติ

## อยากต่อยอด?

- ตั้งงบแต่ละหมวดเองในแอป
- เพิ่มหลายสกุลเงิน (THB/USD)

- ลง App Store / Play Store จริง (ต้องเขียนใหม่ด้วย React Native/Flutter)
